# WhereNow - Family Safety & Location Tracking

A modern Android mobile application for family safety and real-time location tracking.

## Features

- 👑 Host (Admin) features: create/manage groups, invite members, view live locations, track history
- 👤 Member features: join groups, share location, send SOS alerts
- 👶 Simplified Child Mode with one-tap safe mode
- 🗺️ Real-time GPS tracking and map view
- 🆘 SOS emergency system
- 📍 Geofence alerts
- 🔐 Role-based permission system
